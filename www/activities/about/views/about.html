About Waistline

Waistline was created by David Healey with suggestions and contributions from users.

Waistline was built using Cordova available under the Apache license. Also included are Framework7, jQuery, jQuery localize, and ChartJS which are under the MIT license. The HTML, CSS, and other javascript that make up the main body of the app are under the GPLv3 license.

The source code for Waistline can be found at github.com/davidhealey/waistline.
